• 배열이란?
  •  변수들의 모임
  •  배열 선언, 초기화, 대입
  •  배열 리터럴( [] )를 이용하는 방법
  • var array = [] ;
  •  배열 요소 CRUD(create/read/update/delete)
  •  추가 / 읽기 / 수정 / 삭제
  •  배열 요소 열거
  •  for 를 사용하는 경우
  •  for in 을 사용하는 경우
  •  배열 프로퍼티와 메서드

  •  sort() : 배열의 요소의 값을 정렬
